# delayed_input_optimization
Code accompanying the paper "Efficient optimisation of physical reservoir computers using only a delayed input" [[https://doi.org/10.1016/j.neunet.2023.06.014](https://doi.org/10.1038/s44172-025-00340-6)], written by E. Picco, L. Jaurige, K. Lüdge, S. Massar, and published in Nature Communications Enginering. 

The complete scripts to run the experiments are provided. 

The VHDL code is used to program the Virtex-7 FPGA chip. 

The software interface is in MATLAB.
