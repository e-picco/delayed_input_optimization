# delayed_input_optimization
Code accompanying the paper "Efficient optimisation of physical reservoir computers using only a delayed input" [https://doi.org/10.1016/j.neunet.2023.06.014], written by E. Picco, P. Antonik, S. Massar, and published in Neural Networks.  Experimental code in Python (software interface) and VHDL (to program the Virtex-7 FPGA chip).
